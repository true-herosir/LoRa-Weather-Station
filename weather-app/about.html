Weather Monitoring System

Our state-of-the-art Weather Monitoring System leverages cutting-edge IoT technology and LoRaWAN infrastructure to deliver precise environmental data monitoring. The system architecture comprises distributed sensor nodes that collect real-time atmospheric measurements, seamlessly transmitting data through The Things Network's secure infrastructure.

Key System Features:

  • High-precision environmental sensors for accurate data collection
  • Secure data transmission via LoRaWAN protocol
  • Robust SQL database integration through MQTT messaging
  • Multi-platform accessibility:
    • Responsive web interface
    • .NET MAUI cross-platform application

Developed in collaboration with Saxion University of Applied Sciences, this system represents the convergence of IoT innovation and environmental monitoring technology.
